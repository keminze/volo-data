# VoloData 社区行为准则

## 我们的承诺

为了创建一个开放和欢迎的环境，作为贡献者和维护者，我们承诺：

- 使参与我们社区的每一个人都获得无骚扰体验，无论其年龄、身体特征、残障状况、种族、民族、性征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教信仰或性取向如何。
- 采取明确的行动创建包容性环境。
- 尊重不同的观点和经验。
- 对批评持开放态度。
- 关注对社区最有利的事情。
- 向社区成员表现出同情。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 对不同的观点和经验尊重
- 优雅地接受建设性批评
- 关注对社区最有利的事
- 表现出对他人的尊重和同情

不可接受的行为包括：

- 使用性化的语言或意象
- 人身攻击、贬低或污辱性评论
- 公开或私下骚扰
- 未经明确许可公布他人的隐私信息
- 其他不适当的专业行为

## 我们的责任

社区主持人负责澄清可接受行为的标准，并对不可接受行为采取适当的纠正措施。

社区主持人有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献，并在必要时临时或永久禁止任何他们认为不适当、具有威胁性、冒犯性或有害的贡献者。

## 执行

违反此行为准则的情况可以通过以下方式报告给负责执行的项目维护者：

- 发送电子邮件至: kmz3225147671@gmail.com
- 私下报告给 GitHub 用户（如果可用）
- 在我们的社区论坛上报告

所有投诉都将被审查和调查，并根据情况进行适当处理。项目团队有义务为事件报告者保持机密。

## 适用范围

本行为准则适用于：

- 项目空间（代码库、问题跟踪、讨论、PR）
- 代表项目的官方活动
- 公共空间（如线上论坛、社交媒体）
